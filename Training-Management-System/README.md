# Training Management System

## Description
A Salesforce Apex project demonstrating business logic, DML operations,
triggers, and asynchronous Apex using a training-based use case.

## Concepts Covered
- Business Classes
- DML Operations (Insert, Update, Delete)
- Trigger & Trigger Handler pattern
- Asynchronous Apex (Future, Queueable, Batch)
- Test Classes with 99% code coverage

## Business Classes
- Car.cls
- Mobile.cls
- Rectangle.cls
